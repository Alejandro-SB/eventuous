# Eventuous

Event Sourcing library for .NET, which is:
- Somewhat volatile as breaking changes keep coming in
- Very opinionated
- [EventStoreDB](https://eventstore.com)-oriented
- .NET-friendly (decent DI, logging, and diagnostics support)

Authors:
